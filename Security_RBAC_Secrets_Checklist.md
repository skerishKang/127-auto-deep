# 보안/권한/비밀 관리 체크리스트

- [ ] 모든 시크릿은 Vault에 저장, 코드/환경 파일에 직접 노출 금지
- [ ] 워크스페이스/프로젝트 별 RBAC 적용 및 최소 권한 원칙 준수
- [ ] 시크릿 만료 정책/회전 주기 설정(예: 90일)
- [ ] 배포 전 시크릿 유효성 자동 검증
- [ ] 승인 워크플로 적용(고위험 템플릿/프로젝트)
- [ ] 감사 로그(생성/조회/회전/삭제) 활성화
- [ ] 공급자 폴백/백오프 설정 및 비용 쿼터
- [ ] 알림: 실패율>5%, p95 지연>3분, 비용 급증>30%/일
- [ ] 정기 취약점 스캔/권한 리뷰(월 1회)
- [ ] 사고 대응: 키 폐기/폴백/복구 절차 문서화
